<h2>一、项目简介</h2>
基于SSM的在线医疗服务系统

角色：管理员、用户
<h2>二、项目技术</h2>
<p class="md-end-block md-p"><span class="md-plain">后端：SSM(Spring+SpringMVC+Mybatis)</span></p>
<p class="md-end-block md-p"><span class="md-plain">前端：JSP+CSS+JS+JQUERY+Layui</span></p>

<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">

<strong>用户</strong>
登陆注册、医生坐诊、预约挂号、药品、药品购买、健康资讯、新闻资讯、我的、跳转到后台、我的地址、收藏、购物车、客服

<strong>医生</strong>
主页、个人中心、医生坐诊管理、药品管理、预约挂号管理、用户病例管理

<strong>管理员：</strong>
主页、个人中心、科室管理、医生管理、医生坐诊管理、药品分类管理、药品管理、用户管理、预约挂号管理、用户病例管理、健康资讯管理、系统管理、订单管理

</div>
